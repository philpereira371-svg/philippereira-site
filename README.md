# philippereira.com — founder one-pager

Self-contained static site. No build step.

## Deploy
Drag this folder into Netlify (or `netlify deploy`), or connect this repo to
Netlify/Vercel. Then point `philippereira.com` (IONOS) at the host.

## Before going live
- Replace the two `https://www.linkedin.com/` links with the real LinkedIn URL.
- Turn on `phil@philippereira.com` email forwarding at IONOS (the contact button uses it).
- Optional: add a real headshot.

Files: `index.html` (page), `og-image.png` (share card), `netlify.toml` (headers).
